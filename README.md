# Salat-Time-esp32
.الهدف من المشروع تصميم ساعة رخيصة الثمن وقابلة للتخصيص مستقبلا مثل عرض أحاديث أو أذكار أو مواقيت المحاضرات

مميزات المشروع:
تكلفة منخفضة
قابل لتغيير الشكل في أي وقت
سهل النفل والتركيب
